## Traveling Salesperson Problem(TSP, 외판원문제)
  * 여러 도시들과 각 도시들로 이동하는 비용이 주어졌을때 출발도시에서 모든도시들을 한번씩 방문하고 출발도시로 돌아오는 최소경로를 찾는 문제

## Genetic Algorithm(GA, 유전알고리즘)
  * 유전변이법칙에서 유래된 알고리즘
